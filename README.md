# 📌 SafeFlow – Plataforma de Análise de Transporte de Cargas Perigosas

Entrega de uma interface interativa que reúne os principais dados sobre movimentação de cargas perigosas, a partir das bases do IBAMA. O produto permite uma análise simplificada e intuitiva das informações, como volume movimentado, mapeamento Origem-Destino e aplicação de filtros para enriquecer a análise.

Pretende-se também identificar as principais empresas movimentadoras e os tipos de produtos transportados.

<a name="projeto"></a>
# 📊 Projeto

Este projeto apresenta uma interface interativa para visualização de dados de cargas perigosas, com foco em:

- Centralizar dados de movimentação;
- Integrar informações para análise estratégica;
- Mapear Origem-Destino das cargas;
- Aplicar filtros que enriquecem a análise;
- Facilitar a fiscalização e a gestão de grandes volumes de dados.

<a name="objetivo"></a>
# 🎯 Objetivo do Projeto

O objetivo é desenvolver um painel interativo que permita:

- Visualização clara dos dados de transporte de cargas perigosas;
- Apoio à tomada de decisão logística;
- Identificação de padrões e riscos;
- Melhoria na análise e fiscalização.
